Thư mục này dành cho các asset 3D chung. Với chế độ xây cầu,
hãy đặt mô hình tùy chỉnh của bạn vào `public/bridge-models` (ví dụ
`custom_bridge.glb`). Nếu bạn đổi tên file, cập nhật hằng số `DEFAULT_MODEL_PATH`
trong `components/Physics/Bridge3D/BridgeModel.tsx` cho phù hợp.

File nên ở định dạng GLB/GLTF để react-three-fiber có thể nạp được.
